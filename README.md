# Volt Logistics - Telegram WebApp

Веб-приложение для системы заказов Volt Logistics.

## 🚀 Возможности

- 📦 Оформление заказов из Китая и Японии
- 📸 Загрузка и сжатие фотографий товаров
- 🧮 Калькулятор стоимости доставки
- 💱 Актуальные курсы валют
- 📱 Полная интеграция с Telegram Bot

## 🛠 Технологии

- HTML5, CSS3, JavaScript
- Telegram WebApp API
- Автоматическое сжатие изображений
- Адаптивный дизайн

## 📋 Структура проекта

```
├── index.html          # Главная страница
├── order.html          # Форма заказа
├── calculator.html     # Калькулятор доставки
├── currency.html       # Курсы валют
├── reviews.html        # Отзывы
├── profile.html        # Профиль пользователя
├── script.js           # Основная логика
├── style.css           # Стили
└── img/               # Изображения и иконки
```

## 🔧 Настройка

1. Загрузите файлы на GitHub Pages
2. Настройте Telegram Bot с URL вашего GitHub Pages
3. Обновите URL в коде бота: `webapp_url = "https://yourusername.github.io/your-repo/"`

## 📱 Использование

Приложение работает через Telegram Bot. Пользователи получают доступ через кнопку WebApp в боте.

## ✨ Особенности

- **Автоматическое сжатие изображений** до 800x600 с качеством 70%
- **Оптимизация данных** для Telegram API лимитов
- **Адаптивный интерфейс** для мобильных устройств
- **Валидация форм** с подробными сообщениями об ошибках
